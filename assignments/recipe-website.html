Assignment: Recipe Website

Part 1

Using semantic HTML, build a webpage that displays a favourite recipe of yours. This webpage should include:

  • A title
  • A brief description of the recipe
  • A list of ingredients
  • A series of steps to perform the recipe
  • Subheadings as appropriate
  • At least one image
  • At least one link

Part 2

Use CSS to style your recipe page as follows:

  • Centre all heading tags (<h1>, <h2>, etc.) using the text-align property
  • Change the font family of the heading tags to something you like
  • Change the font family of the rest of the text on the page to something you like (hint: you can apply CSS rules to the whole <body>)
  • Make all links (whether visited or not) the same colour, and change their appearance to not have an underline using the text-decoration property
  • Add a background-color to the page's title, and also change the color of the page's title text
  • Scale the image to fit within 250 px (pixels), using the height and/or width properties
  • Position the image on the right of the page using the float property, and have the text flow around it
